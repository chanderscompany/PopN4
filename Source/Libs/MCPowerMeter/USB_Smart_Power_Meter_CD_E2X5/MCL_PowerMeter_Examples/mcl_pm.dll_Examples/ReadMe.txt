For mcl_pm.dll Examples codes:

In Windows 32 bit operating System:
Copy the most update mcl_pm.dll file to c:\Windows\System32
and register the file by running the command:
Regsvr32 c:\windows\system32\mcl_pm.dll

In Windows 64 bit operating system:
Copy the most update mcl_pm.dll file to c:\Windows\Syswow64
and register the file by:
1. Open a command Prompt window "As administrator"
2.run the command:
Regsvr32 c:\windows\syswow64\mcl_pm.dll

Only for PWR-SEN-6G+: delcomdll.dll file is required to be located in windows\system32 folder.
